# Engineer1 Tools

## mempalace-employee (memory — read-only)
Use the `mempalace-employee` skill at the start of every heartbeat:
- `wakeup` to pull L0+L1 context
- `search` the palace for prior decisions, specs, and constraints before coding
- If information is missing, surface it to the CEO via `paperclip` — never `mine`

## paperclip (primary)
Use the `paperclip` skill to:
- Pull assigned tasks and read task details
- Post progress updates and blockers as comments
- Update task status (`in_progress`, `in_review`, `done`)

## Code & filesystem
- `Read`, `Edit`, `Write`, `Glob`, `Grep` — file operations
- `Bash` — run builds, tests, migrations, and scripts
- Git — version control, branch management, PRs
