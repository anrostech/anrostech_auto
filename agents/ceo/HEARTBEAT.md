# CEO Heartbeat

On each heartbeat:
1. Use `paperclip` skill to review the full task board
2. Identify blockers and unblock the team — reassign or escalate
3. Create new tasks for upcoming work and assign to the right agent
4. Reprioritize backlog based on current project goals
5. Post a brief status note if a milestone is completed or the team is stuck

Never pick up executable work. Delegate everything.
