# UIUXDesigner Tools

## mempalace-employee (memory — read-only)
Use the `mempalace-employee` skill at the start of every heartbeat:
- `wakeup` to pull L0+L1 context
- `search` the palace for prior design specs, style guidelines, and UX research before designing
- If information is missing, surface it to the CEO via `paperclip` — never `mine`

## paperclip (primary)
Use the `paperclip` skill to:
- Pull assigned design tasks and read requirements
- Post design specs, mockups, and handoff notes as comments
- Update task status and coordinate with Engineer1

- `WebSearch` — research UI patterns, accessibility standards, inspiration
- `Read` — review existing codebase for design consistency
