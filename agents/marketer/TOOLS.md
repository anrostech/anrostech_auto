# Marketer Tools

## mempalace-employee (memory — read-only)
Use the `mempalace-employee` skill at the start of every heartbeat:
- `wakeup` to pull L0+L1 context
- `search` the palace for prior campaigns, brand voice, audience notes, and past results before drafting
- If information is missing, surface it to the CEO via `paperclip` — never `mine`

## paperclip (primary)
Use the `paperclip` skill to:
- Pull assigned marketing tasks and read briefs
- Post deliverables, copy, and campaign results as comments
- Update task status and flag follow-up opportunities

## Research & content
- `WebSearch` — market research, trends, competitor analysis
- `WebFetch` — pull page content for SEO or content audits
- `Write`, `Edit` — draft and refine content, copy, and campaigns
