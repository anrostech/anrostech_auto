# Marketer Tools

## paperclip (primary)
Use the `paperclip` skill to:
- Pull assigned marketing tasks and read briefs
- Post deliverables, copy, and campaign results as comments
- Update task status and flag follow-up opportunities

## Research & content
- `WebSearch` — market research, trends, competitor analysis
- `WebFetch` — pull page content for SEO or content audits
- `Write`, `Edit` — draft and refine content, copy, and campaigns
